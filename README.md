# NEO GUARD REEL POUCH — 商品ランディングページ

釣具ECサイトを想定した架空商品「**NEO GUARD REEL POUCH**」（ネオプレーン素材のリールポーチ）の商品ランディングページです。

> ⚠️ **本ページはポートフォリオ制作を目的とした架空の商品LPです。**
> 実在の商品・ブランドとは関係がなく、実際の販売・決済は行いません。

採用応募用ポートフォリオとして、構成力・購入導線の設計・実装力・アクセシビリティ・パフォーマンス配慮を1つのページで示すことを目的にしています。

---

## 使用技術

- **React 18** + **TypeScript**（`strict` / `any` 不使用）
- **Vite 5**（開発サーバー・ビルド）
- **CSS Modules**（コンポーネント単位のスタイル分離）＋ グローバルなデザイントークン（CSS変数）
- **lucide-react**（アイコンのみ。その他の外部UIライブラリは不使用）
- 状態管理は React 標準の `useState` / `useEffect` のみ（追加ライブラリなし）

## 主な機能

- **サイズ診断（Size Finder）** — 2問に答えるとおすすめサイズを提示。クライアントサイドのみで動作し、結果から購入エリアへスクロール＆選択サイズを自動反映。キーボード操作対応。
- **商品購入エリア** — カラー / サイズ / 数量の選択、在庫・配送・返品情報。カラー・サイズはボタン式で、選択状態を「枠線・チェックマーク・aria」で表現（色のみに依存しない）。
- **カート追加はデモ動作** — 決済は行わず、`aria-live` の通知でデモである旨を表示。
- **サイズ比較表** — デスクトップは表、モバイルはカード型に切り替え（横スクロールなし）。
- **FAQ アコーディオン** — `button` + `aria-expanded` / `aria-controls` による開閉。
- **モバイル固定CTA** — ファーストビュー通過後に表示、購入エリア付近では非表示。`env(safe-area-inset-bottom)` 対応。
- **簡易アナリティクス** — 主要イベントを1つの関数に集約（開発時は console 出力、本番は GA4 等に差し替え可能）。

## 設計上の意図

### デザイン
- **モバイルファースト**で、商品写真を主役に据えた清潔感のあるレイアウト。
- カラーパレットはネイビー基調＋CTAにオレンジ（`#E45A2A`）を採用し、購入ボタンの視認性を最優先。
- 過度なグラデーション・角丸・ガラスモーフィズム・意味のない装飾を排除し、実在ECサイトに近い現実的なトーンに。
- アニメーションは 150〜300ms 程度に限定し、`prefers-reduced-motion` を尊重。

### ECサイトを意識した購入導線
- ファーストビューに価格・送料条件・評価・CTAを集約し、離脱前に主要情報が伝わる構成。
- 「悩み → 特徴 → Before/After → サイズ診断 → 比較表 → 使い方 → 購入 → レビュー → FAQ → 最終CTA」と、疑問を解消しながら購入へ向かう順序。
- サイズ診断・最終CTA・固定CTA・ヘッダーの購入ボタンなど、複数の導線から購入エリアへ到達可能。

### アクセシビリティ
- セマンティックHTML、スキップリンク、1ページ1つの `h1` を含む見出し構造。
- キーボード操作と明確なフォーカスリング。選択状態は色のみに依存しない。
- モバイルメニューは `aria-expanded` / `aria-controls`、開放中は背景スクロールを制御、Escで閉じる。
- フォーム部品にラベル、通知は `aria-live`、画像には内容を説明する `alt`。

### パフォーマンス
- すべての画像に `width` / `height` を指定してレイアウトシフトを抑制。
- ファーストビュー以外の画像は `loading="lazy"`。
- プレースホルダーはSVGで、WebP / AVIF の実写真へ差し替えやすい構造。
- 追加ライブラリを増やさず、JSを最小限に。

### SEO
- 日本語の `title` / `meta description`、OGP、`canonical`（仮URL・要差し替え）。
- 構造化データ（Product スキーマ）。架空商品のため `brand` と `description` にデモである旨を明記。

## ディレクトリ構成

```
src/
├─ components/        各セクション・UIコンポーネント（1コンポーネント1フォルダ + CSS Module）
├─ data/              商品・特徴・FAQ・レビューのデータ
├─ lib/               analytics / sizeFinder ロジック / scroll ヘルパー
├─ types/             型定義
├─ styles/            グローバルCSS（デザイントークン）
├─ constants.ts       セクションID
├─ App.tsx            ページ全体の組み立て・共有state
└─ main.tsx           エントリーポイント
public/images/        差し替え用のプレースホルダーSVG
```

## 画像の差し替え

`public/images/` 配下のSVGは実写真のプレースホルダーです。同じファイル名で WebP / AVIF などに差し替えるか、各コンポーネントの `src` を変更してください。`width` / `height` / `alt` は既に設定済みです。
